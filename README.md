# Student_Cheongwon
* 김은학 개인 프로젝트

## 프로젝트 소개
 국민들이 국가기관에 무언가를 요청하거나 건의하고 싶을 때는 국민 청원 사이트를 이용합니다. 
 
 여기에서 영감을 얻어서 학생들이 학교에 의견을 낼 수 있는 학생 청원 사이트를 제작하였습니다.

 해당 웹사이트에서 제공하는 기능은 다음과 같습니다.

 1. 회원가입 및 로그인 기능
 2. 자신의 의견을 담을 수 있는 게시물 및 댓글 작성 기능
 3. 다른 의견들에 찬성 혹은 반대할 수 있는 기능
